# HTM Annotator

A Next.js application that allows you to overlay `.htm` files, highlight text, and save the annotated files.

## Features

- **Upload**: Load `.htm` or `.html` files.
- **Highlight**: Select text and highlight it (yellow background).
- **Save**: Download the annotated HTML file.

## Tech Stack

- Next.js
- Tailwind CSS
- shadcn/ui
- Lucide React

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
